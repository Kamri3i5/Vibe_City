# 🌆 VibeCity — Где сейчас движ? (Ташкент)

VibeCity — это интерактивная "живая" карта города, которая помогает жителям и гостям Ташкента находить самые активные локации или, наоборот, спокойные места для отдыха в реальном времени.

![VibeCity Preview](https://via.placeholder.com/800x400?text=VibeCity+Live+Map) <!-- Можно заменить на скриншот позже -->

## ✨ Основные фишки
- **Живая карта:** Визуализация активности города на базе Leaflet.
- **Вайб-чеки:** Ставьте отметки (🔥 Кайф, 💀 Глухо, 😭 Скучно), чтобы поделиться атмосферой.
- **Алгоритм Decay (Затухание):** Информация всегда актуальна — старые голоса теряют вес со временем.
- **Индикатор загруженности:** Примерная оценка количества людей в локации.
- **Фильтрация:** Быстрый поиск по категориям (Еда, Парки, Развлечения и т.д.).
- **Тёмная тема:** Удобно использовать вечером.

## 🛠 Технологии
- **Frontend:** HTML5, Vanilla CSS, JavaScript.
- **Карты:** [Leaflet.js](https://leafletjs.com/) + CartoDB Dark Matter.
- **Иконки:** [Lucide Icons](https://lucide.dev/).
- **Шрифты:** Inter (Google Fonts).

## 🚀 Как запустить локально
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Kamri3i5/Vibe_City.git
   ```
2. Откройте `index.html` в вашем браузере.
   *Совет: Используйте расширение "Live Server" в VS Code для автоматического обновления страницы при сохранении.*
